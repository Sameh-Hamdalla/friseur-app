# 💈 Hamdalla Booking System

Ein einfaches Online-Buchungssystem für Services (z. B. Friseur), gebaut mit React und n8n.

---

## 🚀 Features

- 📅 Datum auswählen (Kalender)
- 💇 Service auswählen
- ⏰ Uhrzeit wählen
- 🛒 Übersicht (Warenkorb)
- ✅ Buchungsbestätigung
- 📧 Automatische E-Mail über n8n

---

## 🛠️ Technologien

- React
- CSS (modular strukturiert)
- n8n (Webhook + Automation)
- ngrok (lokale Entwicklung)
